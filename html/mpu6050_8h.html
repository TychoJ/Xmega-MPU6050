Detailed Description

MPU6050 library for the Xmega with the Xmega-TWI library

Author
Tycho Jobsis
Date
16-04-2018
Version
0.1.0

This library is used to communicate with an MPU6050 sensor over I2C.

Note
For the HvA-Xmegaboard you need to use TWIE!
This library does not support slave devices for the mpu6050!!!
